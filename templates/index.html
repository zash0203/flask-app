Welcome to My Flask Web Page

Installation Process

I installed Flask, a lightweight Python web framework, using the following command in the terminal:

pip install Flask

Building the Page

To create this simple web page, I followed these steps:

  1. Created a Flask app in Python (app.py).
  2. Defined a route for the home page and rendered an HTML template.
  3. Created an HTML template (index.html) inside a 'templates' folder.
  4. Styled the page using basic HTML and optional CSS.
  5. Run the Flask app using the development server.

The content of this page is minimal, but you can expand it based on your needs.
